### Building

1. Clone the repo
   ```sh
   git clone https://github.com/devinschubert14/2d-render-engine-opengl
   ```
2. Make build directtory
   ```sh
   cd 2d-render-engine-opengl && mkdir build
   ```
3. Create cmake build files
   ```sh
   cmake ..
   ```
4. Build with cmake
   ```sh
   cmake --build .
   ```
